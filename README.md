# ApexPlanet Task 5 - Final Project

## Project Overview

This project was completed as part of the ApexPlanet Data Analytics Internship Program. The objective was to perform data analysis, statistical modeling, automation, and reporting using the Superstore dataset.

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* SciPy
* Jupyter Notebook

## Tasks Completed

### 1. Data Cleaning & Preparation

* Cleaned and prepared the dataset for analysis.
* Checked for missing values and data consistency.

### 2. SQL Analysis

* Performed SQL queries for business insights.
* Used filtering, grouping, aggregation, and sorting operations.

### 3. Dashboard Development

* Created visual dashboards to analyze sales, profit, and customer trends.
* Generated charts and KPI-based insights.

### 4. Advanced Analytics

* Statistical Analysis (Mean, Median, Mode, Standard Deviation)
* Hypothesis Testing (T-Test)
* Customer Segmentation using K-Means Clustering
* Predictive Modeling using Linear Regression

### 5. Automation & Reporting

* Developed an automation script using Python.
* Calculated key business KPIs automatically.
* Generated a final executive report.

## Key Results

* Total Sales: 2,326,534.35
* Total Profit: 292,296.81
* Identified customer segments using clustering techniques.
* Built a predictive model for sales forecasting.
